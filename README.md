# 📊 Supermarket Sales Analysis Using SQL

## 📌 Project Overview
This project analyzes supermarket sales data using SQL. The objective was to query, filter, aggregate, and analyze sales data to identify top-performing products and cities.

---

## 🛠 Tools & Technologies
- SQL  
- SQLite Online  

---

## 📂 Dataset
The dataset contains:
- Product  
- Price  
- Quantity  
- City  

---

## 🔍 SQL Skills Demonstrated
- SELECT statements  
- WHERE filtering  
- GROUP BY aggregation  
- ORDER BY sorting  
- LIMIT ranking  

---

## 📊 Key Insights
- 🥇 Eggs generated the highest total sales  
- 📍 London had the highest overall revenue  
- 📉 Birmingham had the lowest total sales
